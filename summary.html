IT Infrastructure and Emerging Technologies

IT Infrastructure

IT infrastructure is the shared technology resources that provide the platform for the firm's specific information system applications. IT infrastructure includes hardware, software, and services that are shared across the entire firm.

The stages of IT infrastructure evolution

The five stages of IT infrastructure evolution are:

Mainframe era.

Personal computers era.

Client/Server era.

Enterprise Computing era.

Cloud and Mobile computing era.

Technology Drivers of Infrastructure Evolution

Moore's Law:Moore's Law deals with the exponential increase in processing power and decline in the cost of computer technology, stating that every 18 months the power of microprocessors doubles, and the price of computing falls in half.

The Law of Mass Digital Storage: This law deals with the exponential decrease in the cost of storing data, stating that the number of kilobytes of data that can be stored on magnetic media for $1 roughly doubles every 15 months.

Metcalfe's Law: this law states that a network's value to participants grows exponentially as the network takes on more members.

Moreover, an increase in computer use is also being fueled by the quick drop in communication costs and increased acceptance of computing and communications standards in the technology sector.

The components of IT infrastructure

IT infrastructure components include: computer hardware platforms, operating system plat- forms, enterprise software platforms, networking and telecommunications platforms, data manage- ment software, Internet platforms, and consulting services and systems integrators.

Current trends in computer hardware platforms

More and more people are using mobile digital platforms for computing.

Quantum computing could dramatically boost processing power through the ability to be in more than one state at the same time.

Virtualization organizes computing resources and it is not restricted by physical configuration or geographic location.

Cloud Computing: Instead of buying and installing the hardware and software on their own computers, businesses and people use cloud computing, which allows them to access computing power and software as services across a network, including the Internet.

Edge computing helps optimize cloud computing by performing some data processing on a set of linked servers at the edge of the network, near the source of the data.

Multi-core processor: which it is a microprocessor with two or more processing cores added for improved performance is referred to as a multi-core processor.

Green Computing: In order to reduce the negative effects on the environment, green computing methods and technologies are used in the production, use, and disposal of information technology hardware.

Current trends in computer software platforms

Open-source software is produced and maintained by a global community of programmers

Java and JavaScript provide software for making web applications more interactive.

HTML5 makes it possible to embed images, audio, and video directly into a web document without add-on programs.

Web services are open web standards-based, loosely connected software components that can be used with any operating system and application software.

Companies are purchasing their new software applications from outside sources, or by renting online software services (SaaS).

Mashups combine two different software services to create new software applications and services.

The challenges of managing IT infrastructure and management solutions

Major challenges including:

Dealing with platform and infrastructure change.

Infrastructure management and governance.

Making wise infrastructure investments.

Solution guidelines include using a competitive forces model

Many firms are turning to cloud computing in an effort to reduce their IT platform costs.
